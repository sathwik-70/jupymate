# **App Name**: Jupymate Navigator

## Core Features:

- Endpoint Display: Display of API Endpoints: Clearly present used API endpoints, configurations, Jupiter ecosystem tools, wallet support list, governance analytics and visual tools references.
- Cross-Token Swaps UI: Display Cross-Token P2P Swaps: Allow users to visualize and execute cross-token peer-to-peer payments via Jupiter's liquidity aggregator.
- Route Visualization: Visualization of Swap Route Analytics: Graphically display swap routes to illustrate how tokens are exchanged via Jupiter.
- Wallet Display: Presentation of Wallet Support: Display compatibility with Phantom, Solflare, Glow, Backpack, Ledger.
- Governance Metrics: Display Governance Data: Present fetched Governance metrics via integration Catalytics, Flipside, Litterbox APIs
- Contextual Assistance: AI-Powered Tooltips with MCP Config: Allow the AI to review configuration files (MCP config) for API endpoints and parameters, and provide appropriate tooltip advice on each.
- Documentation Link: Link to Documentation: Prominently place a link to the complete Jupymate documentation for comprehensive guidance.

## Style Guidelines:

- Primary color: Saturated blue (#4287f5), echoing Solana's vibrant tech landscape.
- Background color: Very light desaturated blue (#f0f4ff).
- Accent color: Saturated violet (#a564c7) providing contrast.
- Headline font: 'Space Grotesk' (sans-serif) to reflect the app's computerized focus; body font: 'Inter' (sans-serif) for longer texts and optimal legibility.
- Use clean, outline-style icons from Lucide to visually represent API tools, wallet functions, governance actions.
- Implement a card-based layout for API tools and documentation links. Employ consistent padding, margins, and spacing for visual grouping.
- Apply subtle transition effects (e.g., fade-ins, slide-ups) to highlight newly-fetched data or UI elements.
